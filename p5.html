Processors: CPU vs GPU

‚öôÔ∏è CPU

Central Processing Unit - Brain of computer

  • Cores: 4-16 powerful cores typical
  • Speed: 2-5 GHz per core
  • Best for: Sequential logic, control
  • Latency: Optimized for single-task speed
2024 Examples:

Apple M4 (10 cores), Intel Core Ultra (14 cores), Snapdragon 8 Gen 3 (8 cores)

üéÆ GPU

Graphics Processing Unit - Parallel processor

  • Cores: 1000s of small cores
  • Throughput: Processes 1000s operations simultaneously
  • Best for: Gaming, AI, video encoding
  • Power: 80-450W depending on model
2024 Examples:

NVIDIA RTX 4090 ($1600), H100 ($35K), Apple GPU (8-20 cores)

Aspect CPU GPU
Architecture Few powerful cores Thousands small cores
Speed Focus Individual task speed Total throughput
Power Draw 35-55W typical 80-450W depends
Best For General computing Gaming, AI, video

Trained using 10,000-20,000 NVIDIA H100 GPUs. Cost: $350-700M in hardware alone. Why? CPU training would take 50+ years. GPU: a few months. This shows why AI companies need GPU acceleration.
