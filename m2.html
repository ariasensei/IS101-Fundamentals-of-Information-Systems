Understanding the fundamentals of operating systems

What is an Operating System?

An operating system (OS) is system software that acts as an intermediary between users and computer hardware. It manages all hardware and software resources and provides a user-friendly interface for interacting with the computer.

ğŸ“– Key Concepts

Process: A running instance of a program. Each process has its own memory space and resources.
Thread: A lightweight subprocess that shares memory with its parent process. Multiple threads can run in parallel.
Multitasking: The ability to run multiple programs seemingly at the same time (through rapid switching).
User Interface: Can be Command-Line Interface (CLI) or Graphical User Interface (GUI).

ğŸ’¾ Resource Management

How OS manages computer resources

âš¡ CPU Scheduling

The OS scheduler decides which process gets to use the CPU and for how long.

FIFO (First In First Out)

Processes execute in the order they arrive. Simple but can cause long waits.

Round Robin

Each process gets a small time slice (quantum). Fair and prevents starvation.

Priority-Based

Higher priority processes get CPU time first. Flexible and responsive.

Shortest Job First

Process with shortest execution time runs first. Minimizes average wait time.
