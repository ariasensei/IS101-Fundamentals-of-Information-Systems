What is a Computer Network?

A computer network is a collection of interconnected devices (computers, servers, printers, smartphones) that can communicate and share resources with each other.

ğŸ“– Key Terminology

Bandwidth: The maximum amount of data that can be transmitted over a network in a given time (measured in Mbps or Gbps).
Latency: The time delay for data to travel from source to destination (measured in milliseconds).
Throughput: The actual amount of data successfully transferred over a network in a given time.

ğŸ“œ Protocols & Models

Rules that govern network communication

What are Network Protocols?

Network protocols are standardized rules that define how data is formatted, transmitted, received, and acknowledged between devices on a network. Think of them as the "language" devices use to communicate.

ğŸŒ Common Network Protocols

HTTP/HTTPS

HyperText Transfer Protocol - Used for web browsing. HTTPS adds encryption for security.

TCP/IP

Transmission Control Protocol/Internet Protocol - The foundation of internet communication.

FTP

File Transfer Protocol - Used for uploading/downloading files between computers.

SMTP/POP3/IMAP

Email protocols - SMTP sends mail, POP3/IMAP receive mail.

DNS

Domain Name System - Translates domain names (google.com) to IP addresses.

DHCP

Dynamic Host Configuration Protocol - Automatically assigns IP addresses to devices.

ğŸ“š The OSI Model (7 Layers)

The Open Systems Interconnection model describes how data moves through a network in 7 layers:

7

6

5

4

End-to-end data delivery (TCP, UDP)

3

Network Layer

Routing and IP addressing (IP, ICMP)

2

Data Link Layer

Node-to-node data transfer (Ethernet, Wi-Fi)

1

Physical Layer

Physical connection (cables, signals, bits)
