What is a Spreadsheet?

A spreadsheet is a software application that displays and organizes data in rows and columns. It allows users to enter, calculate, analyze, and visualize data efficiently using formulas and built-in functions.

ğŸ”§ Key Spreadsheet Components

Cell: The basic unit of a spreadsheet where data is entered. Identified by column letter and row number (e.g., A1, B5).
Row: A horizontal line of cells, numbered from top to bottom (1, 2, 3...).
Column: A vertical line of cells, labeled with letters (A, B, C...).
Worksheet: A single sheet containing rows and columns. A file can have multiple worksheets.
Workbook: The entire file that contains one or more worksheets.

What is a Formula?

A formula is an expression that performs calculations on values in cells. It always begins with an equals sign (=) and can contain cell references, operators, and functions.

â• Basic Mathematical Operators

+

Addition: = A1 + B1

-

Subtraction: = A1 - B1

*

Multiplication: = A1 * B1

/

Division: = A1 / B1

^

Exponent: = A1 ^ 2

%

Modulus: = A1 % B1

ğŸ“š Common Functions

SUM()

Adds up all numbers in a range.

=SUM(A1:A10)

AVERAGE()

Calculates the average of numbers in a range.

=AVERAGE(B1:B10)

COUNT()

Counts the number of cells with numerical values.

=COUNT(C1:C20)

IF()

Returns one value if condition is true, another if false.

=IF(A1>100,"Pass","Fail")

MAX() & MIN()

Find the largest and smallest values in a range.

=MAX(D1:D10) | =MIN(D1:D10)

ğŸ“ Understanding Cell References

Relative Reference: A1

Changes when copied. Copying from A1 to B1 changes it to B1.

Absolute Reference: $A$1

Stays the same when copied. The $ signs lock the cell.

Range Reference: A1:A10

Refers to multiple cells from A1 to A10.
